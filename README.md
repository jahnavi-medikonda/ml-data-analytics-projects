# Machine Learning & Data Analytics Projects

## Overview
This repository contains hands-on projects demonstrating data analysis, machine learning modeling, and end-to-end data workflows using Python and SQL.

These projects focus on solving real-world problems using structured datasets, applying machine learning techniques, and generating actionable insights.

---

## Projects

### 1. Machine Learning Models (ICP5 – ICP9)
- Built classification and regression models using Logistic Regression, Decision Trees, Random Forest, and SVM  
- Performed data preprocessing including missing value handling, feature scaling, and encoding  
- Evaluated models using accuracy, RMSE, and cross-validation  
- Applied hyperparameter tuning (Grid Search, Random Search) to improve performance  

---

### 2. Data Analysis & Python Workflows (ICP1 – ICP4)
- Developed Python scripts for data processing and automation  
- Performed exploratory data analysis using Pandas and NumPy  
- Identified trends, correlations, and patterns in datasets  
- Created visualizations to support insights
